# Open Source Status

TatruMedGemma was created for the Kaggle MedGemma Impact Challenge. The
challenge submission period is over.

## Why The Repository Remains Public

The repository remains public for:

- learning and code reading
- reproducibility of the prototype architecture
- experimentation with local, LAN, and hosted inference modes
- responsible community continuation after the competition

## Post-Deadline Changes

Minor stability fixes were applied after the challenge deadline on
February 25, 2026 to keep the repo and demo paths functional. No new model
claims, competitive re-submission claims, or Kaggle evaluation claims are
made for those post-deadline changes.

## What This Does And Does Not Mean

- It does mean the repository can continue to receive bugfixes,
  documentation updates, safety clarifications, and maintenance work.
- It does not mean there is ongoing Kaggle endorsement, official challenge
  evaluation, or continuing competition status.

## Contribution Direction

Contributors are encouraged to improve:

- documentation quality
- setup reliability
- public-repo hygiene
- privacy and safety clarity
- reproducibility for research/demo purposes

Contributors should not reposition the repository as a clinically validated
or healthcare-ready product.
