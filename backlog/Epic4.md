# Epic 4: Archived Medical And Emergency Ideas

## Archive Note

This file records challenge-era brainstorming only. These ideas are not
current product claims and should not be interpreted as approved medical or
emergency functionality.

## Overview

Possible future concepts for user-controlled escalation, summary sharing, and
accessibility support.

## Historical User Stories

### US4.1: Generate Conversation Summary

**As a** user,  
**I want** to generate a concise summary of a chat,  
**So that** I can review or share it with another person if I choose.

### US4.2: Show Local Emergency Contact Information

**As a** user,  
**I want** the app to display local emergency contact information,  
**So that** I can reach official local services directly if needed.

### US4.3: Share Geolocation

**As a** user,  
**I want** location sharing to be an explicit, user-controlled action,  
**So that** I can decide whether to provide it to another person or service.

### US4.4: Text-to-Speech Assist

**As a** user,  
**I want** the app to read responses aloud,  
**So that** I can listen instead of reading.

## Historical Progress

- Summary generator UI not yet built
- No local emergency database integrated
- Geolocation code not surfaced in the current app
- Basic TTS button implemented in chat bubbles

## Boundary

These archived ideas do not make the repository an emergency tool, medical
device, or clinically suitable system.
