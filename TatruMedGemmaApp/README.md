# TatruMedGemma App

This directory contains the Expo / React Native client for TatruMedGemma.

TatruMedGemma is a research and demonstration prototype created for the
Kaggle MedGemma Impact Challenge. It is not a medical device and must not be
used for diagnosis, treatment, emergency care, or unsupervised patient-facing
use.

## Quick Start

```sh
npm install
npx expo start
```

The app can be configured for:

- on-device GGUF inference
- LAN Ollama or Flask endpoints
- hosted OpenAI-compatible APIs
- hosted Gradio / Kaggle endpoints

See the repository root [README](../README.md) for setup, privacy, safety,
and licensing notes.
